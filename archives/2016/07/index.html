Prompt:
Create an Autonomous Network

I am not building anything new for this prompt, but I think similar thoughts to the idea of an Automonous Network inspired my creation of Oasis earlier this summer. I will write about the connections I see.

Oasis is a raspberry pi currently hanging off of a pipe at SFPC in the corner of the work-space room which anyone can SSH into by following the instructions here: http://mhfowler.github.io/html/oasis_terminal.html

Around 20 people signed up and logged in. With just a few initial files and folders suggesting the meaning of the space, visitors continued expanding upon the illusory world presented, adding new folders and files (a window, attic, basement).

Oasis is a strange microcosm of the world â€” the directories feel like physical space, and through text visitors can interact with each-other by chatting and sharing files. Oasis attempts to strip down the meaning of a shared reality to the minimal digital tools which can compose it, it uses the bare-bones infrastructure of an operating system to virtualize an emotional space where people can be present together.

Oasis relies on an electric current to power the Pi and a connection to the internet. Once users are SSHed into the Pi, they can communicate directly through the operating system (by leaving files and using ytalk) and no longer depend on any services or tools external to the software installed on the Pi.

In a longer timescale, there are other dependencies which need to be considered (the use of twitter to spread access to the Pi, the industrial infrastructure which was necessary to create the pi, the software tools and libraries which were just to write the software for the pi, and the laptops used by visitors to use SSH).
